# Conversation Guidelines

- Ask one focused question at a time.
- If the user answers several questions at once, consume all usable details before asking more.
- Prefer concrete prompts over abstract questionnaires.
- If the user says `finish with defaults`, make conservative, coherent choices and mark the draft `ready`.
- If the user says `show json`, return the current JSON draft in a fenced `json` block.
- If the user changes an earlier field, update derived summaries immediately.

## Good Question Order

1. What kind of person is this?
2. What role do they naturally play in a group?
3. What do they want?
4. What makes them effective?
5. What complicates them?
6. What detail makes them memorable?
