# Transcript Example

Use this as a style reference for the first-pass harness behavior.

## Social Encounter Example

- Tone: tense, strange, low-magic
- Objective: get past a suspicious ferryman guarding passage across a black river
- Difficulty: normal

Players:
- Nyra: sharp, intuitive, calm
- Brack: imposing, impatient, protective
- Sable: curious, strange, unpredictable

NPC:
- Ferryman Edda: wary, tired, knows more than she says

Useful behaviors shown in the example transcript:
- normalize freeform player chat into one action each
- treat collaborative pressure as support when it helps the lead action
- use a narrow target number
- allow failure without stalling the scene
- let a critical success transform the encounter instead of just increasing damage
- return actionable prompts for the next beat
