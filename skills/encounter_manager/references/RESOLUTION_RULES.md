# Resolution Rules

## Approaches

- `force`
- `finesse`
- `charm`
- `insight`
- `weird`
- `support`

## Difficulty Targets

- `easy` -> `8`
- `normal` -> `12`
- `hard` -> `16`
- `deadly` -> `18`

Keep practical results inside `8` to `18`.

## Modifiers

Use modifiers sparingly.

Preferred range:
- `-2`
- `-1`
- `0`
- `+1`
- `+2`

## Outcome Bands

- natural `1` -> usually `critical_failure`
- natural `20` -> usually `critical_success`
- below target by `3` or more -> `failure`
- below target by `1` or `2` -> `mixed_success`
- meet or exceed target -> `success`
- exceed by `4` or more -> `critical_success`

## Recommended Conditions

Players:
- wounded
- shaken
- cornered
- inspired
- exhausted
- marked
- cursed

NPCs:
- suspicious
- frightened
- persuaded
- hostile
- exposed
- indebted

Monsters:
- staggered
- enraged
- disoriented
- pinned
- weakened
