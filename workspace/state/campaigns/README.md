# Campaigns

Create one folder per durable campaign.

Naming:

- prefer stable slugs like `ash-market`
- do not rely on one chat transcript as the identity key

Minimum required files:

- `campaign.json`
- `players.json`
- `npcs.json`
- `clocks.json`
- `logs/event-log.jsonl`
- `handoffs/latest-session.json`
